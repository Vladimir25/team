Фильтрация
===

Условия фильтрации отправляются в виде GET-параметров.

Можно задать фильтрацию по значению атрибута:

```json
{
		"parent_id": 12
}
```

Или сразу по нескольким атрибутам:

```json
   {
   		"parent_id": 12,
   		"category_id": 21
   }
   ```

При этом, условия буду склеены оператором `AND`.
